# Cap height

## Definition
Cap height is the distance from the baseline to the top of the uppercase letters.

## Why it matters
Cap height measures flat-topped capitals like "H" and "T". Rounded or pointed caps like "O" and "A" usually overshoot it slightly, which is intentional so they look the same size optically. Cap height is taller than the [x-height](x-height.md) but not necessarily the same as the height of [ascender](ascender.md)s, and it is the top reference that Figma's [vertical-trim](vertical-trim.md) uses when it trims line spacing.

## Example
In most typefaces the top of an "H" sits at the cap height.
