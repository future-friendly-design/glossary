# Ethiopic script

## Definition

Ethiopic is an abugida used for several languages of Ethiopia and Eritrea, where each character is a consonant and vowel combined into a single block.

## Why it matters

Ethiopic (also called Ge'ez, after the classical language it was first used for) is an [abugida](../language-terms/writing-systems-and-scripts/abugida.md), but it is structurally distinct from the [Brahmic](brahmic-scripts.md) abugidas covered elsewhere in the glossary. Instead of attaching a separate vowel sign to a base consonant, each consonant-plus-vowel combination is a single character whose shape is modified to show the vowel, so one base consonant has a set of related forms, one per vowel. That makes it sit somewhere between an abugida and a [syllabary](syllabary.md) in feel. The practical consequence: covering Ethiopic means supporting a large set of distinct characters, not a small alphabet plus a few marks, so font coverage and text input both differ from a Latin or Brahmic baseline.

## Example

A single Ethiopic consonant appears in a row of related shapes, one for each vowel, rather than taking a detachable vowel mark.
