# Georgian script

## Definition

The Georgian script is an alphabet used to write Georgian; its modern everyday form is called Mkhedruli and has no separate uppercase.

## Why it matters

The Georgian script is an [alphabet](alphabet.md) with separate consonant and vowel letters written left to right, and like [Armenian](armenian-script.md) it is a single-language script: essentially one language, its own letterforms, easy to overlook in coverage planning. One feature worth knowing for interface work is that the modern form used for everyday writing, Mkhedruli, is unicameral: it does not have the uppercase and lowercase distinction that Latin, Greek, and Cyrillic have, so styling that assumes a capital form (small caps, automatic title casing) has nothing to act on. There are older Georgian script forms, but Mkhedruli is the one current text uses.

## Example

Georgian written in Mkhedruli has a single set of letters with no capital forms, so a "capitalize each word" rule simply does nothing.
