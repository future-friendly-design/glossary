## Definition
ISO 639-3 is a standard that gives a unique three-letter code to every known human language.

## Why it matters
ISO 639-3 aims for comprehensive coverage of all individual languages, including living, extinct, ancient, and constructed ones, going far beyond the smaller ISO 639-1 (two-letter) set most developers know from web `lang` attributes. [SIL Global](sil.md) maintains it as the registration authority, and the standard sorts identifiers into three scopes: individual languages, [macrolanguage](macrolanguage.md)s, and collections. It is the backbone of language metadata in linguistics, libraries, and software, so it is often the precise way to say which language a string, font, or locale is for when the two-letter codes run out.

## Example
The code "cmn" identifies Mandarin Chinese and "eng" identifies English.
