## Definition
Katakana is the angular Japanese syllabary used mainly for foreign loanwords and emphasis.

## Why it matters
Katakana is the second Japanese [[syllabary]], covering the same set of syllable sounds as [[hiragana]] but with sharper, more angular shapes. It is used chiefly for words borrowed from other languages, scientific names, onomatopoeia, and for emphasis, roughly the way italics work in Latin text. Like hiragana it appears together with kanji in everyday writing, so a Japanese interface has to set all three scripts comfortably on the same line.

## Example
The loanword "coffee" is written in katakana as ko-hi.
