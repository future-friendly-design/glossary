# Logographic

## Definition

A logographic script uses symbols that stand for whole words or meaningful units rather than for sounds alone.

## Why it matters

In a logographic writing system each character represents a word or morpheme, so the symbol carries meaning rather than mapping directly to a single sound. Chinese characters are the most widely used logographic system, though they also encode pronunciation in part. For type and software work the practical consequence is scale: a logographic script needs thousands of distinct characters rather than a few dozen letters, which raises real questions about font file size, glyph coverage, and input methods.

## Example

The Chinese character for "tree" represents the whole word, not the individual sounds in it.
