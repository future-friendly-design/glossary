# Logosyllabary

Logosyllabary (also morphosyllabary) is a more precise name for a [logographic](logographic.md) system such as Chinese: each character stands for a word or morpheme and at the same time for a syllable, which a plain "logographic" label leaves out. See [logographic](logographic.md) for the full entry.
