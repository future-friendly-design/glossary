## Definition
The Unicode Standard is the universal character encoding standard that assigns every character in the world's writing systems a unique code so text displays consistently across devices.

## Why it matters
Maintained by the Unicode Consortium (founded 1991), the Unicode Standard assigns a unique code point to each character across the world's scripts so text can be stored, exchanged, and displayed reliably regardless of platform. It also publishes the code charts and annexes that define text direction, normalization, segmentation, and how characters combine, the documents this glossary cites throughout. As a reference source it is the foundational one for typography, fonts, and any software that handles multilingual text. (For the concept itself, see [unicode](unicode.md).)

## Example
The letter "A" is the same Unicode code point (U+0041) whether typed on a phone, a laptop, or a website.
